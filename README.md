# gittrace

a clean desktop app to watch your git projects in real time.

---

## what it does

- **live file watching** — sees changes the moment they happen
- **workspace** — browse files, edit, view diffs, markdown preview
- **character-level diffs** — highlights exactly what changed, not just which lines
- **side-by-side compare** — pick any two commits and diff them visually
- **push overview** — see what's local, what's pushed, what's behind
- **insights** — commit activity, authors, hot files

## install

download the latest release for windows:

- [`Gittrace.exe`](https://github.com/Smudio/Gittrace/releases/latest) — portable, no install
- [`Gittrace-Setup.exe`](https://github.com/Smudio/Gittrace/releases/latest) — installer

or build it yourself:

```
git clone https://github.com/Smudio/Gittrace.git
cd Gittrace
npm install
npm start
```

## shortcuts

| key | action |
|---|---|
| `1`–`7` | switch tabs |
| `ctrl+f` | search commits |
| `r` | refresh |
| `ctrl+s` | save file |
| `esc` | close modal |
